# Descrição

Este documento descreve o conjunto de atributos de metadados que são considerados relevantes no âmbito da [ETSI](https://www.etsi.org/). São retirados de duas especificações relevantes a nível europeu, nomeadmente, o _DCAT Application profile for data portals in Europe_ ([DCAT-AP](https://op.europa.eu/en/web/eu-vocabularies/dcat-ap)) e o _Infrastructure for Spatial Information in Europe_ ([INSPIRE](https://knowledge-base.inspire.ec.europa.eu/index_en)). O primeiro é uma especificação baseada no vocabulário do Catálogo de Dados do W3C ([DCAT](https://www.w3.org/TR/vocab-dcat/)) para: (1) descrever conjuntos de dados do sector público Europeu, (2) permitir a pesquisa de conjuntos de dados entre portais de dados e (3) tornar os dados do sector público mais facilmente pesquisáveis entre fronteiras e sectores. O segundo é uma infraestrutura europeia de dados espaciais para apoio às políticas ambientais comunitárias, com enfoque na partilha transfronteiriça de dados e que fomenta a interoperabilidade de serviços para apoio aos vários destinatários (decisores e executores políticos, empresas, cidadãos, entre outros).

## Notas

A definição do modelo de dados no catálogo nacional de dados é um processo contínuo, podendo surgir alterações ao longo do tempo, que devem de ser incorporadas nos sistemas em produção.
Quando as propriedades INSPIRE `Geographic Bounding Box` e `Metadata Date` existirem em simultâneo com as propriedades DCAT `geographical coverage` e `modification date`, deve ser assegurada a compatibilidade entre os conteúdos de forma a garantir a qualidade da informação.
