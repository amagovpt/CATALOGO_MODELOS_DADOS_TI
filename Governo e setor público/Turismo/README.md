# Descrição

Os modelo de dados a seguir para representar dados relacionados com a temática **Turismo** são os disponibilizados pelo [Turismo de Portugal](https://dadosabertos.turismodeportugal.pt), nomeadamente os respeitantes às categorias de [Alojamento Turístico](https://dadosabertos.turismodeportugal.pt/search?categories=%252Fcategories%252Falojamento%2520tur%25C3%25ADstico), [Equipamento, Infraestruturas e Atividades Turísticas](https://dadosabertos.turismodeportugal.pt/search?categories=%252Fcategories%252Fatividades%2520tur%25C3%25ADsticas) e [Ordenamento Turístico](https://dadosabertos.turismodeportugal.pt/search?categories=%252Fcategories%252Fatividades%2520tur%25C3%25ADsticas).

Para todos os modelos disponibilizados pelo Turismo de Portugal existe metainformção associada, disponível quando se seleciona *Visualizar detalhes completos*. Por exemplo, para o conjunto de dados Estabelecimentos de Alojamento Local, pode ser consultada [aqui](https://dadosabertos.turismodeportugal.pt/datasets/turismoportugal::estabelecimentos-de-alojamento-local/about) a metainformação disponível.

## Notas

A definição do modelo de dados no catálogo nacional de dados é um processo contínuo, podendo surgir alterações ao longo do tempo, que devem de ser incorporadas nos sistemas em produção. Além disso, o modelo permite a inclusão de atributos e de metainformação específica para determinados verticais. No entanto, esses atributos podem ser ignorados quando há integração de dados provenientes de várias entidades, sendo apenas usados os atributos aqui descritos.
