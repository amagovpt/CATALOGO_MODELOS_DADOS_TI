# ℹ️ Sobre este projeto

## Missão

O Catálogo Nacional de Modelos de Dados para os Territórios Inteligentes visa promover a **interoperabilidade, harmonização e qualidade** dos dados disponibilizados pelas entidades públicas portuguesas, no contexto do programa [Territórios Inteligentes](https://territoriosinteligentes.gov.pt).

## Entidade responsável

Este catálogo é mantido pela **ARTE — Agência para a Modernização Administrativa**.

## Formato e licenciamento

- Os modelos de dados são documentados em **Markdown**, garantindo legibilidade e fácil versionamento.
- O conteúdo está disponível em **português europeu (PT-PT)**.
- O repositório é público e está disponível em [github.com/amagovpt/CATALOGO_MODELOS_DADOS_TI](https://github.com/amagovpt/CATALOGO_MODELOS_DADOS_TI).

## Estado do projeto

![Estado: em evolução](https://img.shields.io/badge/status-em%20evolu%C3%A7%C3%A3o-blue)

Este é um instrumento **vivo**, em permanente atualização e expansão, à medida que novos domínios e necessidades das entidades públicas são identificados.

## Como contribuir

1. Consulte o [repositório no GitHub](https://github.com/amagovpt/CATALOGO_MODELOS_DADOS_TI).
2. Abra uma *issue* para reportar problemas ou sugerir melhorias.
3. Submeta um *pull request* com propostas de alteração aos modelos existentes.

## Contacto

Para questões relacionadas com o catálogo, utilize a secção de [Issues](https://github.com/amagovpt/CATALOGO_MODELOS_DADOS_TI/issues) do repositório oficial.
