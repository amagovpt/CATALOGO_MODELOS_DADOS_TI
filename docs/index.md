## 🎯 O que é este catálogo

O **Catálogo Nacional de Modelos de Dados para os [Territórios Inteligentes](https://territoriosinteligentes.gov.pt)** é um instrumento fundamental para promover a **interoperabilidade**, a **harmonização** e a **qualidade** dos dados disponibilizados pelas entidades públicas.

Organiza e documenta modelos de dados normalizados, garantindo uma referência comum para a **publicação, partilha e reutilização** de informação a nível nacional.

Foi desenvolvido para apoiar as entidades na **adoção de práticas consistentes**, assegurando que os dados publicados seguem padrões uniformes — tanto a nível estrutural como semântico. Ao fazê-lo, contribui para:

- 🔗 **Interoperabilidade** entre sistemas e entidades
- 📐 **Harmonização** estrutural e semântica dos dados
- ✅ **Qualidade** consistente da informação publicada
- 🚀 **Eficiência operacional** e melhoria dos serviços digitais
- 🔍 **Transparência** na disponibilização de dados públicos

## 🚀 Como utilizar

1. Aceda à área temática relevante para a sua entidade ou caso de uso.
2. Consulte o modelo de dados normalizado correspondente.
3. Adeque os seus dados à estrutura e semântica definidas no modelo.
4. Em caso de dúvida, consulte as **[FAQs](./faq/)**.
