# Descrição

O modelo de dados a seguir para representar **Transportes públicos**
é o [Transmodel](https://transmodel-cen.eu). Este modelo é um [standard europeu](https://www.en-standard.eu/csn-en-12896-1-public-transport-reference-data-model-part-1-common-concepts/) para lidar com informação de transportes públicos multi-modais. Este modelo encontra-se dividido em várias partes, das quais se destacam:

- [NetEX](https://transmodel-cen.eu/index.php/netex/), desenhado para troca de informação de transportes públicos, tais como paragens, itinerários, horários e tarifas;
- [SIRI](https://transmodel-cen.eu/index.php/siri/), para troca de informação de transportes públicos em tempo real.

Os [XML Schemas](https://github.com/NeTEx-CEN/NeTEx) estão disponíveis num repositório de código abertyo, existindo [documentação extra ](https://data4pt.org/) e uma [ferramenta de validação](https://greenlight.itxpt.eu) dos modelos.
